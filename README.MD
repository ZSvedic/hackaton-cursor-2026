## Overview

[DOWNLOAD CekLess-demo-1.mov](CekLess-demo-1.mov)

ČekLess is an intuitive tool designed to simplify healthcare appointment scheduling in Zagreb, tackling the long waiting lists and complexity of the current system (CEZIH – Central Health Information System of Croatia).  

---

## Problem
Scheduling medical appointments in Zagreb can be slow and complicated due to long queues and fragmented provider information.  

---

## Solution
ČekLess provides a **clear overview and dataset of healthcare providers** offering appointments for a selected medical exam on the **earliest available date**.  

**User Inputs:**
- Desired medical exam  
- Preferred examination date  

**Workflow:**
1. User selects the medical exam.  
2. User optionally selects a preferred date.  
3. ČekLess retrieves the earliest available appointment from hospital forms.  

> "We can’t solve all your problems, but we can at least solve this one 🙂"  

---

## Limitations
The current solution only addresses appointment scheduling and does not cover other healthcare management needs.
